---
title: Create a New Project
description: How to create a documentation project in WiseSpark.
---

## Steps

<Steps>
  <Step title="Log in">
    Go to `https://app.wisespark.com` and sign in.
  </Step>
  <Step title="Start new project">
    Click "New Project" on the dashboard.
  </Step>
  <Step title="Fill project details">
    Enter project name and description, then click "Create".
  </Step>
  <Step title="Add your first page">
    Select "Add Page" to start documenting.
  </Step>
</Steps>
